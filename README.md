# Nombre del Proyecto

> Aplicación web desarrollada para el Trabajo Práctico Evaluativo de Práctica Profesionalizante I (JavaScript) y (Python).

## 📖 Descripción

Este proyecto consiste en una aplicación web desarrollada con **HTML, CSS y JavaScript**, cuyo objetivo es consumir una API desarrollada en Python, permitiendo visualizar, crear, editar y eliminar registros de forma dinámica.

Además, incorpora el uso de **LocalStorage** para almacenar elementos seleccionados por el usuario, implementando una vista de favoritos o carrito.

El desarrollo cumple con los requisitos establecidos para el **Trabajo Práctico Evaluativo 01 de JavaScript**.

---

# Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- LocalStorage
- API REST (FastAPI / Python)

---

# Funcionalidades

## Visualización de datos

- Obtener todos los registros mediante una petición **GET**.
- Consultar el detalle de un elemento por su ID.
- Mostrar la información dinámicamente en una única página.

## Gestión de datos

- Crear nuevos registros (**POST**).
- Modificar registros existentes (**PUT**).
- Eliminar registros (**DELETE**).

## LocalStorage

- Agregar elementos a favoritos.
- Eliminar elementos de favoritos.
- Persistencia de datos mediante LocalStorage.

## Interfaz

- Diseño responsive.
- Navegación sencilla.
- Actualización dinámica del contenido.

# Requisitos del trabajo práctico

El proyecto cumple con los siguientes requisitos:

- Consumo de API mediante Fetch.
- Peticiones GET.
- Peticiones POST.
- Peticiones PUT.
- Peticiones DELETE.
- Visualización dinámica de datos.
- Uso de LocalStorage.
- Implementación de eventos.
- Separación de la lógica en funciones reutilizables.
- Desarrollo utilizando una única página (`index.html`).
